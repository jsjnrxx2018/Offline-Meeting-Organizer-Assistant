# 会议信息录入助手

一个本地 Windows 桌面小工具，用于把会议通知文本解析成标准字段，并自动写入 Excel 会议安排表。

## 功能

- 可视化界面：会议文本输入、解析结果预览、提示词模板展示。
- 支持固定格式输入，也支持部分自然语言描述。
- 支持一次粘贴多条会议。
- 可选择任意 `.xlsx` 文件，并记住上一次使用的 Excel 路径。
- 写入后自动按日期和开始时间排序。
- 相同日期自动合并日期列。
- 自动统一会议数据区字体、边框、对齐和换行格式。
- Excel 已打开时可实时写入；未打开时可后台写入文件。

## 运行源码版

1. 安装 Python。
2. 安装依赖：

```powershell
pip install -r requirements.txt
```

3. 启动程序：

```powershell
python meeting_assistant.py
```

也可以双击：

```text
run.bat
```

## 输入示例

推荐使用标准格式：

```text
会议时间：2026年5月8日 14:30
会议名称：科研工作推进会议
会议地点：综合楼301会议室
负责部门：科研处
出席范围：各学院负责人
召集人：张三
出席领导：李四
```

也支持较自然的单句描述：

```text
下周三上午九点在实验楼208会议室召开实验室安全检查整改会，与会者：实验室管理处 科研处 教务处 保卫处 各学院实验中心，召集人是周伟副校长。
```

自然语言解析依赖规则匹配，添加前建议先点击“解析预览”检查结果。

## Excel 表格要求

默认表头为第 3 行，字段顺序为：

```text
日期 | 开始时间 | 会议名称和内容 | 地点 | 负责部门 | 出席范围 | 召集人 | 出席领导
```

仓库中提供了一个默认模板：

```text
会议安排.xlsx
```

## 打包

打包说明见：

```text
PACKAGING.md
```

已验证可使用 PyInstaller 生成 Windows 可执行程序。正式安装包可用 Inno Setup 6 根据 `installer/会议信息录入助手.iss` 编译。

## 注意

- 实时写入已打开 Excel 需要 Windows + Microsoft Excel。
- 后台写入 `.xlsx` 不要求 Excel 打开。
- 用户配置会保存到本机用户目录，不会写入程序安装目录。
